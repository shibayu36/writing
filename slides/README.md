## スライド用の画像を作る for Nano Banana Pro
```
# 命令書
プレゼンテーションスライドの視覚補助となる、主役にならないシンプルなイラストを作成してください。以下のスタイルを厳守してください。

# スタイル指定
* **表現:** フラットデザイン、ミニマリストなベクターイラスト。アイコンのように単純な幾何学図形で構成し、影やグラデーション、複雑な質感は一切排除する。
* **文字の禁止:** 画像内には、いかなるテキスト、文字、数字、記号も一切含めないこと。これは絶対条件である。
* **配色:** メインの色を鮮やかな青（カラーコード #1e88e5 に近い色）とし、それらに合う薄い青、および白の3〜4色程度に限定する。
* **構図:** 純粋な白背景。資料のテキストを邪魔しないよう、余白（ネガティブスペース）をたっぷりと取った、すっきりとした配置。

# 描画する内容の説明
[ ここに、具体的に描きたいモノや概念を、視覚的な言葉で入力する ]
```
